# Roadmap

本路线图展示 0.1.x 至 0.3.x 版本的规划，帮助社区参与与追踪进度。

## v0.1.x
- [x] 项目骨架与文档
- [x] 样例数据与分层课程
- [x] CLI & API 最小可用功能

## v0.2.x
- [ ] 向量数据库接入（Qdrant/Milvus）
- [ ] LLM 集成：支持 OpenAI、DeepSeek、通义千问等推理接口
- [ ] 文献自动摘要与标签生成流水线
- [ ] 前端原型：基于 Next.js 或 SvelteKit 的可视化门户

## v0.3.x
- [ ] 可视化知识图谱与课程规划器
- [ ] 与 LMS 平台对接（Moodle、钉钉等）
- [ ] 数据更新自动化调度（Airflow/Celery）
- [ ] 多语言支持（中/英）

## 长期目标
- 建立开放社区，鼓励教师、学生、研究者共建资源。
- 引入自动评估体系衡量学习成效与资源质量。
- 构建跨平台 API，服务教育、科研与企业应用场景。
